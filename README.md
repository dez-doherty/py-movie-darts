# Movie Darts

## Description

A python-based party game where players name films matching a chosen category and try to check out first

## Requirements

- `pygame`
- `pygame_gui`
- `requests`

## Technologies

- `Python`
- `OMDb API`

## Features

- `Pulls live film data (Box Office, Runtime, IMDb Rating, Metascore, Awards) from the OMDb API`
- `Category-based guessing across Genre, Director, Writer, and Actors`
- `Darts-style scoring with bust rules, checkout range, and per-visit caps`
- `Multi-player turn system with the ability to quit mid-game and auto-declare remaining winners`
- `Setup, game, and end screens themed through a shared pygame_gui theme`
- `Packaged as a standalone desktop executable via PyInstaller`
